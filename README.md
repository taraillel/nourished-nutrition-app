# Nourished – AI Nutrition & Dietary Compliance Platform

## Overview
Nourished is an AI-based nutrition app focused on ingredient-level analysis and dietary compliance.

Most nutrition apps track calories. Nourished focuses on what’s actually inside the food and whether it fits a user’s dietary restrictions.

## Problem
Existing apps don’t work well for people with dietary restrictions or chronic conditions.

They miss:
- ingredient-level detail
- support for multiple diets at once
- clear feedback on whether a meal is actually compliant

Users are left reading labels, guessing ingredients, and making decisions without reliable guidance.

## Solution
Nourished analyzes food across different input types and evaluates it against a user’s dietary framework.

The goal is to make it easy to understand whether a meal is compatible with specific dietary needs.

## Features

### Food Input
Users can log meals through:
- images (meal photos)
- voice input
- text input

### Ingredient Analysis
- breaks meals down into ingredients  
- flags problematic components (additives, seed oils, sweeteners, etc.)  
- focuses on food quality, not just macros  


### Diet Frameworks
Supports multiple diets, including:
- gluten-free  
- dairy-free  
- low FODMAP  
- low histamine  
- anti-inflammatory  
- keto / low carb  
- plant-based  

Users can combine multiple restrictions at once.


### Compliance Scoring
- evaluates how well a meal fits selected diets  
- highlights conflicts  
- suggests basic adjustments  


### Tracking
- daily meal logging  
- basic nutrition breakdown (calories, protein, fiber, sugar)  
- trend tracking over time  


### Restaurant Use (Planned)
- analyze menus  
- suggest compliant options  
- recommend modifications  


### AI Assistant
- chat-based interface  
- answers questions about ingredients and diets  
- helps users make decisions in real time  


## Tech
- built using no-code / AI-assisted tools (Base44)  
- exploring computer vision and language models for food analysis  

## Status
Work in progress. Early prototype.

## Direction
Long term, this can become:
- a nutrition compliance engine  
- a B2B API for health platforms, clinics, and food services
